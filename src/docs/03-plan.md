## 1. 修改概述

基于 v2 实现，根据原始设计稿进行高精度还原优化。重点修复：侧边栏样式、顶部信息栏、设备屏幕模块、执行步骤模块、实时日志模块、底部统计卡片、整体布局。

### 1.1 视觉风格

| 属性 | 值 |
|------|-----|
| 主色调 | #2563FF |
| 背景色 | #F5F7FB |
| 卡片背景 | #FFFFFF |
| 边框色 | #E5E7EB |
| 卡片圆角 | 16px |
| 卡片阴影 | 0 1px 2px rgba(0,0,0,.04) |

风格：浅色科技风 + 极简控制台风格，参考 Linear/Vercel Dashboard/Raycast
